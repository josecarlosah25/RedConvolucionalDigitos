
Para poder correr la GUI se hace uso de eel, por lo que es necesario instalarlo mediante: 

$sudo apt install python-pip 
$sudo pip3 install eel

Puede requerir otras dependencias como 

$sudo pip3 install wheel
$sudo pip3 install setuptools

Finalmente se debe correr dentro del mismo directorio en el que se encuentra, para que la GUI funcione, ya que es basado en web. Solo se debe ejecutar el archivo main.py

$python3 main.py
